# File Renamer Tool

## Описание
Скрипт для массового переименования файлов в указанной директории. Заменяет указанный текст в именах файлов на новый.

## Функциональность
- Массовое переименование файлов
- Замена указанного текста в именах файлов
- Простой интерактивный интерфейс
- Обработка ошибок и исключений

## Требования
- Python 3.6+
- Модуль `signature` (должен быть в той же папке)

## Установка

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd file-renamer
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

## Использование

### Запуск скрипта
```bash
python renamer.py
```

### Пример использования
```
Введите путь: C:\Users\Username\Documents\photos
Введите старое название: IMG_
Введите новое название: Photo_
```

### Что делает скрипт:
- Ищет все файлы в указанной папке
- Находит файлы, содержащие указанный текст
- Заменяет старый текст на новый в именах файлов
- Выводит результат переименования

## Обработка ошибок
- При нажатии `Ctrl+C` выводится сообщение "Прервано пользователем"
- При завершении программы выполняется очистка ресурсов

## Завершение программы
1. Нажмите `Ctrl+C` в консоли
2. Или закройте окно консоли
3. Программа автоматически завершится с сообщением "Очистка ресурсов..."

## Создание исполняемого файла (EXE)

### Установка PyInstaller
```bash
pip install pyinstaller
```

### Создание EXE файла
```bash
pyinstaller --onefile renamer.py
```

### Готовый EXE файл
После сборки EXE файл будет находиться в папке `dist/renamer.exe`

## Особенности
- Работает с любыми типами файлов
- Сохраняет расширения файлов
- Не изменяет содержимое файлов, только имена
- Поддерживает кириллицу и спецсимволы

## Ограничения
- Требует прав на запись в указанную директорию
- Не переименовывает скрытые файлы и системные файлы
- Не работает с файлами, которые используются другими процессами

## Безопасность
- Всегда делайте backup файлов перед массовым переименованием
- Проверяйте результаты на тестовой папке
- Убедитесь, что у вас есть права на переименование файлов
